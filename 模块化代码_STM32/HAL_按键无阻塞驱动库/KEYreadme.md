# 按键检测模块（KEY）

该模块实现了对多种按键事件的检测与识别，包括单击、双击、长按、重复按等，适用于嵌入式系统（如 STM32）中的GPIO按键扫描逻辑。

## 文件结构

* **KEY.c**：按键状态机实现，包含按键状态读取、状态更新、事件标志设置等。
* **KEY.h**：按键数量、按键码、事件标志等宏定义以及函数声明。

## 特性

* 支持多按键（默认最多支持4个）
* 支持以下按键事件：

  * `KEYDOWN`：按下瞬间
  * `KEYUP`：松开瞬间
  * `KEYHOLD`：持续按住
  * `KEYSINGLE`：单击
  * `KEYDOUBLE`：双击
  * `KEYLONG`：长按
  * `KEYREPEAT`：长按期间的重复触发

## 使用方法

1. 在主循环中周期性调用 `Key_Tick()`（推荐周期为1ms）
2. 通过 `Key_Check(uint8_t key, uint8_t flag)` 检查按键事件是否触发，例如：

   ```c
   if (Key_Check(KEY_1, KEYSINGLE)) {
       // KEY_1 单击事件
   }
   ```

## 配置说明

* 宏定义：

  * `KEY_COUNT`：设置支持的按键数量（默认4）
  * `KEY_1 ~ KEY_4`：按键编号
* 修改按键读取函数 `GetKey_State()` 中的 `HAL_GPIO_ReadPin()` 以适配实际硬件引脚。

## 注意事项

* 本模块假设所有按键为低电平按下（`GPIO_Read == 0`）
* 所有状态在 `Key_Tick()` 中被更新，确保按时调用建议在定时器里使用，搭配1ms定时通过20倍频实现软件消抖
