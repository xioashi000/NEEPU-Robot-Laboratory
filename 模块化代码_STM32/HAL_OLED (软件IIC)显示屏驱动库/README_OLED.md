# 移植于江科大的 0.96 寸 OLED_HAL 显示屏驱动库

## 1. 简介和功能

本库移植于江科大的 OLED_HAL 显示屏模拟 IIC 驱动库，基于 STM32F103C8T6 和 0.96 寸 OLED 显示屏，使用 HAL 库进行开发。该库支持多种显示功能，包括绘制点、线、矩形、圆形、文本等，并且支持多种颜色和字体大小。此外，该库还支持图像显示， 配合取模使用。

## 2. 移植步骤与使用说明

1. 将 OLED.c 和 OLED.h 文件添加到工程中
2. 在 cube_mx 里将 SCL 和 SDA 引脚配置为 GPIO 推挽输出
3. 将 OLED.c 的引脚改成 cube_mx 配置的引脚
4. 初始化

   ```c
   OLED_Init(); // 初始化 OLED
   ```

5. 更新显示

   ```c
   OLED_Update(); // 更新显示
   ```
