# 模块化代码

## 简介

**东北电力大学智能机器人实验室成员仓库。**
旨在对软硬件进行解耦，提供模块化编程组件，支持快速开发与集成，适用于多种编程语言和应用场景，助力开发者高效实现功能模块化。
目前拥有以下模块：

1. cube_mx 自带外设配置 -2025/5/30
2. HAL_FreeRTOS 函数语句 -2025/5/30
3. HAL_OLED (软件 IIC)显示屏驱动库 -2025/5/30
4. HAL_OLED (软件 IIC)显示屏驱动库 -2025/5/30
5. HAL_MCP4725(软件 IIC)DAC 驱动库 -2025/5/30
6. HAL_TB67H450 直流电机驱动库-2025/6/2
7. HAL_W25Q64 (软件SPI)FLASH驱动库 -2025/6/2
8. HAL_FreeRTOS 函数语句-2025/6/2
9. HAL_FLASH模拟EEPROM驱动库-2025/6/2
10.HAL_按键无阻塞驱动库-2025/6/3
11.HAL_EC11旋转编码器驱动库-2025/6/2
