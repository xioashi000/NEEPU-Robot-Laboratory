# 🤖 模块化代码

#### 简介

东北电力大学智能机器人实验室成员仓库。

本仓库汇集了我们在机器人开发中常用的**模块化驱动代码**，覆盖三大主流嵌入式平台，拿来即用，告别重复造轮子。

---

#### 📦 目录结构

| 目录 | 平台 | 内容 |
|---|---|---|
| `模块化代码_ESP32` | ESP32 (Arduino) | 外设配置、Arduino 框架下的常用驱动 |
| `模块化代码_MSP430` | MSP430F5529 | 用户 API 手册、LaunchPad 开发指南 |
| `模块化代码_STM32` | STM32 (HAL) | 传感器 / 外设驱动库全家桶 |

---

#### 🔧 STM32 驱动库清单

| 模块 | 说明 |
|---|---|
| EC11 旋转编码器 | 旋转编码器驱动 |
| FLASH 模拟 EEPROM | 片上 FLASH 模拟 EEPROM 存储 |
| FreeRTOS 函数语句 | FreeRTOS 常用模板与封装 |
| MCP4725 DAC | 软件 IIC 驱动 12 位 DAC |
| MPU6050 陀螺仪 | 软件 IIC 驱动 6 轴姿态传感器 |
| MT6816 磁编码器 | 软件 SPI 驱动磁编码器 |
| OLED 显示屏 | 软件 IIC 驱动 OLED 屏 |
| TB67H450 直流电机 | 直流电机驱动 |
| W25Q64 FLASH | 软件 SPI 驱动外部 FLASH |
| 按键无阻塞驱动 | 非阻塞按键检测 |
